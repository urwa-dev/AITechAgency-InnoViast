# AI Tech Agency - Project Based Internship

This is a fully responsive Single Page Landing Website built using HTML5 and Tailwind CSS for the Innoviast Remote Internship Program.

## 🚀 Live Demo
You can view the live website here: https://urwa-dev.github.io/AITechAgency-InnoViast/

## ✨ Features
- Modern AI Dark Theme Layout
- Fully Responsive Design (Mobile, Tablet & Desktop)
- Interactive Navigation & Custom Contact Form
- Built with Tailwind CSS utility classes

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS (via CDN)
