# AI Tech Agency - Project Based Internship

This is a fully responsive Single Page Landing Website built using HTML5 and Tailwind CSS for the Innoviast Remote Internship Program.

### 🚀 Live Demo
You can view the live website here: [https://aitechagency-innoviast.netlify.app](https://aitechagency-innoviast.netlify.app)

### ✨ Features
* **Modern AI Dark Theme Layout:** Aesthetic and professional dark mode design.
* **Fully Responsive Design:** Optimized for Mobile, Tablet, and Desktop screens.
* **Interactive Navigation & Custom Contact Form:** User-friendly interface elements.
* **Built with Tailwind CSS:** Clean and scalable code using utility classes.

### 🛠️ Tech Stack
* HTML5
* Tailwind CSS (via CDN)

### 🚀 Setup Steps
1. **Clone this repository:** 
   `git clone https://github.com/urwa-dev/AITechAgency-InnoViast.git`
2. **Open locally:** 
   Navigate to the folder and open `index.html` in your favorite web browser to view the project.
